# 근태 집계 변환기

기본자료(이벤트 로그) + SO템플릿(부서별, 다중) + 명단 → 해당 월 자동 채움.
**파일은 브라우저 내에서만 처리되고 외부로 나가지 않습니다.**

## 사용
1. https://hyshin6664.github.io/attendance-converter/ 접속
2. 3개 파일 선택 → 변환 실행 → ZIP 다운로드
3. 우측 상단 "📲 설치"로 바탕화면 PWA 설치 가능 (Chrome/Edge)

## 변환 규칙
- 출근 = 발생시각 min, 퇴근 = max
- 근무시간 = 퇴근 - 출근 - 1시간(휴게)
- 비고는 비움 (전부 새로 채움)
- 슬롯 매칭은 SO템플릿의 기존 이름 기준
