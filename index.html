📊 근태 집계 변환기

기본자료 + SO템플릿(부서별 다중) + 명단 → 새 시트(YYYY-MM월)로 자동 생성. 기존 시트는 절대 건드리지 않습니다.

사용법
1) 각 영역에 파일을 드래그&드롭 하거나 파일 선택 클릭
2) 변환 실행 → 결과 ZIP/xlsx 다운로드
3) 결과 파일에는 YYYY-MM월 새 시트가 생성되고, 기존 04월/05월 등 시트는 그대로
· 기본자료는 A·B·C·D 열만 사용 (이벤트 로그)
· SO템플릿은 한 파일에 여러 팀 블록 OK, 여러 파일도 OK
· 새 시트 이름이 이미 존재하면 오류로 중단 (덮어쓰기 방지)

① 입력 파일 (드래그&드롭 가능)

파일을 끌어다 놓거나 우측 버튼
여러 팀 파일을 한 번에 선택/드롭 가능
샘플이라도 OK · 슬롯은 템플릿 기준
