📊 근태 집계 변환기

기본자료 + SO템플릿(부서별 다중) + 명단 → 해당 월 시트(예: 04월)에 직접 작성. 다른 월 시트(01월·02월·03월 등)는 그대로.

사용법
1) 각 영역에 파일을 드래그&드롭 하거나 파일 선택 클릭
2) 변환 실행 → 결과 xlsx/ZIP 다운로드
3) 결과의 04월(해당 월) 시트에 명단 부서 인원 데이터가 채워짐. 기존 외곽선·서식 그대로 유지
· 기본자료는 A·B·C·D 열만 사용 (이벤트 로그)
· 명단의 부서 인원만 작성됨 (명단에 없으면 슬롯 비움)

① 입력 파일 (드래그&드롭 가능)

파일을 끌어다 놓거나 우측 버튼
여러 팀 파일을 한 번에 선택/드롭 가능
샘플이라도 OK · 슬롯은 템플릿 기준

② 이벤트 (선택사항) — 해당 날짜 전 인원 출/퇴/근무 비우고 비고에 텍스트 입력

예: 4월 24일 · 산행 / 4월 30일 · 가정의 날

② 처리 로그

0처리 파일
0처리 인원
0대상 일수
0매칭 누락
