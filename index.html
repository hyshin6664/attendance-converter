📊 근태 집계 변환기

기본자료 + SO템플릿(부서별 다중 가능) + 명단 → 해당 월 자동 채움. 파일은 브라우저 내에서만 처리됩니다.

사용법
1) 기본자료 1개 + SO템플릿 여러 개(부서별) + 명단 1개 선택
2) 변환 실행 → 결과 ZIP 다운로드
3) 우측 상단 📲 설치 누르면 바탕화면 앱처럼 사용 가능 (PWA)
· 명단은 sample이라도 OK — 슬롯은 SO템플릿 기준으로 자동 인식
